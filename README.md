# 小米路由器 SSH 密码计算器 (Xiaomi Wifi Router SSH Password Calculator)

这是一个轻量级、纯前端的网页工具，用于计算小米路由器的 SSH root 密码。

## ✨ 功能特点

* **纯前端计算**：无需部署任何后端服务，直接在本地浏览器中运行，安全可靠。
* **多格式支持**：自动识别标准格式的 SN 码和包含 `/` 符号的特殊 SN 码格式。
* **一键复制**：提供一键复制功能，方便填入 SSH 客户端。
* **无依赖**：原生 JavaScript 实现 MD5 哈希算法，没有任何外部库依赖。

## 🚀 使用方法

1. 将本仓库克隆到本地，或者直接下载 `index.html` 文件。
2. 使用任何现代浏览器（Chrome, Edge, Safari, Firefox 等）双击打开 `index.html`。
3. 找到小米路由器背面的标签，将其中的 **SN 码** 填入页面输入框中。
4. 点击 **“计算密码”** 按钮。
5. 下方会显示 8 位的 SSH 密码，点击 **“复制密码”** 即可到终端中使用。

## 📖 计算原理

小米路由器的默认 SSH 密码是基于硬件的序列号（SN）生成的。本工具使用以下算法：

1. 根据 SN 码是否包含 `/`，选择对应的 Salt（盐值）：
   * 不含 `/` 的标准 SN 使用：`A2E371B0-B34B-48A5-8C40-A7133F3B5D88`
   * 包含 `/` 的 SN 使用另一组特殊倒序盐值：`a05fd2d6-f052-03a4-6e5a-0aa0690bf44d`
2. 将 **SN 码** 和对应的 **盐值** 字符串进行拼接。
3. 对拼接后的字符串计算 MD5 哈希值，得到 32 位的十六进制字符串。
4. 取该 MD5 哈希值的前 8 位字符，即为该路由器的 root 密码。
